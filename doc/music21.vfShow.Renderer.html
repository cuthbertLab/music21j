Class: Renderer

music21.vfShow. Renderer

new Renderer(s, canvas, where)

Renderer is a function that takes a stream, an optional existing canvas element and a DOM element where the canvas element should be placed and renders the stream as Vexflow on the canvas element, placing it then in the where DOM.

"s" can be any type of Stream.

"canvas" and "where" can be either a DOM element or a jQuery object.
